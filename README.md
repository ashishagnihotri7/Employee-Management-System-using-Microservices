# Employee Management System using Microservices

A backend-focused Employee Management System built using **Java 17** and **Spring Boot Microservices**, following a clean and scalable architecture. The project demonstrates how independent microservices communicate using **Spring Cloud Eureka** for service discovery and **OpenFeign** for inter-service communication.

It includes complete **CRUD operations**, **pagination**, **validation**, **exception handling**, **DTOs**, **entity mapping**, and **Spring Data JPA** with **MySQL** for data persistence. The system is designed with separate **Employee** and **Department** services to showcase real-world microservices development and service-to-service communication.

This project is being developed incrementally by implementing enterprise-level features step by step. The current implementation includes **Eureka Server**, **OpenFeign**, and **API Gateway**, while **Apache Kafka**, **Spring Security (JWT)**, **Config Server**, **Docker**, and monitoring tools are planned for upcoming phases.
